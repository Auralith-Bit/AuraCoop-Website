# सहकारी बचत तथा ऋण सहकारी संस्था लि.

A single-page React application for a Nepali cooperative society website.

## Setup

```bash
npm install
npm run dev
```

The dev server starts at `http://localhost:5173`.

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- React 18
- Vite 5
- Hash-based client-side routing
- Inline CSS
